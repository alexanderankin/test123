Official Releases

Official Releases are compiled and packaged by the core release manager of jEdit. Stable releases usually contain many more features, and are generally more bug free than development releases. Development releases happen more often and are intended for users who want to use the latest plugins and features, but are willing to live with the occasional bug.

Daily Builds

Daily builds are compiled and packaged from the, "latest and greatest" code. The dailies are intended for users who are very active in the jEdit mailing lists and want to be more involved with jEdit development. Builds for the development, and stable branches, and some select plugins are available for download on the TellurianRing.com website.

Debian Linux

To install jEdit via Debian Linux apt-get, add the following line to your /etc/apt/sources.list:

deb http://[mirror].dl.sourceforge.net/project/jedit /

Eg.

deb http://switch.dl.sourceforge.net/project/jedit /

Then, just run apt-get update, followed by apt-get install jedit.

Free BSD

FreeBSD users can find install jEdit from the ports collection by running:

cd /usr/ports/editors/jedit
make install

Gentoo Linux

Gentoo Linux users can install jEdit from the portage tree by running:

emerge jedit

RPM Based Linux

If you use apt4rpm, urpmi, or a similar tool with an RPM-based Linux distribution, you can automatically stay up to date with the latest version of jEdit using the JPackage RPM repository.
