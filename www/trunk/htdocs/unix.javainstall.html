To find out which Java virtual machine is best for your Unix flavor, see the compatibility page.

To install jEdit, enter the following in a command line window (xterm, konsole, etc):

cd directory where you downloaded the JAR file
java -jar filename.jar

For example, if you downloaded jedit41install.jar into ~/Downloads, you would enter:

cd Downloads
java -jar jedit41install.jar

If for whatever reason you want to start the installer in text-only mode, specify text as the last parameter on the command line. In text only mode, the installer will not display it's GUI, and instead it will ask questions in the command line window.

Once jEdit is installed, run the jedit command to start jEdit.
