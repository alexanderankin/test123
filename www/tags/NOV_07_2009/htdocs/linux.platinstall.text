<h3>Installing the Debian package via <code>apt-get</code></h3>

<p>To install jEdit via <code>apt-get</code>, add the following
lines to your <code>/etc/apt/sources.list</code>:</p>

<pre>deb http://dl.sourceforge.net/sourceforge/jedit ./
deb-src http://dl.sourceforge.net/sourceforge/jedit ./</pre>

<p>Then, just run <code>apt-get update</code> and <code>apt-get install jedit</code>
or <code>apt-get source jedit</code>.
