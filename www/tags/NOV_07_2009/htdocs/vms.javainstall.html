Note that the jEdit installer only works on VMS in jEdit 4.2pre1 and above. To find out which Java virtual machine to use, see the compatibility page.

To install jEdit, enter aomething like the following in a command line:

java -jar jedit42pre3install.jar

If for whatever reason you want to start the installer in text-only mode, specify text as the last parameter on the command line. In text only mode, the installer will not display it's GUI, and instead it will ask questions in the command line window.

Once jEdit is installed, run the following command to start jEdit:

java -jar jEdit install dir/jedit.jar
