Configuration

All configuration for this plugin is done in the Look and Feel plugin.
Go to Plugins - Plugin Options - Look And Feel, then select "FlatLaf" from the Look and Feel dropdown. You will have the option to select a theme from a list.

In the Themes list, you can select existing themes to use. These are the pre-defined themes:
