Introduction

Introduction

The JakartaCommons plugin packages a bunch of jars from the Apache Jakarta Commons project. For an exact list, type this in the Console Beanshell: 

jEdit.getProperty("plugin.jakartacommons.JakartaCommonsPlugin.jars")

Escape and Unescape Actions

In addition, the plugin provides a couple of actions for escaping and unescaping unicode characters to hexadecimal numbers. These actions work on the selection, or will select the "word" under or just before the caret.

For example, if you had some funny characters like ❤❶⻪❷➗❸ and selected them, then invoked Escape Unicode, you would see them replaced by a unicode hex escaped form: \u2764\u2776\u2EEA\u2777\u2797\u2778. Likewise, By selecting the escaped strings and invoking the Unescape Unicode would bring back the same ❤❶⻪❷➗❸ characters.

In addition, you can save typing by just entering the hexadecimal values before invoking Unescape Unicode, which will "infer" the leading \u if a hexadecimal value is found in the selection or under the caret.
