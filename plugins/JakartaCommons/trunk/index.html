Introduction

The JakartaCommons plugin packages a bunch of jars from the Apache Jakarta Commons project. For an exact list, type this in the Console Beanshell: 

jEdit.getProperty("plugin.jakartacommons.JakartaCommonsPlugin.jars")

In addition, the plugin provides a couple of actions for escaping and unescaping unicode characters to hexadecimal numbers. These actions work on the selection, or will select the word under the caret.
