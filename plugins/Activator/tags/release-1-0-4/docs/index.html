Activator plugin

Author: Brad Mace

Maintainer: Alan Ezust

The Activator plugin gives very convenient access to the features of loading/unloading plugins from inside jedit. This is especially useful as a developer's tool, when developing and testing plugins.

The Activator plugin contains two dockables, an Activator and a Reloader. Each dockable contains the other's main panel as a JTabPane, so the only difference between them is the default starting point. Only one Activator/Reloader instance can exist at any given time (others will be empty), so dock the view you like best.

Activator panel

The Activator panel allows you to load/unload, and activate/deactivate each plugin (for those which have defered activation support). It also shows you the plugins which are not currently loaded at all, in contrast to the Reloader panel, which only shows the loaded plugins.

Reloader panel

The reloader panel lists each loaded plugin as a convenient button. Click a plugin to reload it and all the plugins that depend on it.
  • Green plugins are loaded and active
  • Yellow plugins are loaded, but not yet activated
  • red plugins loaded with errors
