	* jEdit 4.2pre8

initial plugin central release

Source:
	plugins/Activator
	tag: release-1_0_0

Description:
Lets you see if plugins are loaded or activated and lets you manually (un)load and (de)activate plugins.  
There is also a dockable which allows one-click reloading of plugins.
