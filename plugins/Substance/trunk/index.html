Substance Look and Feel

Author: Dale Anson
Version: 7.4.7.1.00, Oct 2017

This plugin provides the Substance Look and Feel to jEdit. Substance is perhaps the best looking look and feel for Java applications.

Configuration

All configuration for this plugin is done in the Look and Feel plugin.
Go to Plugins - Plugin Options - Look And Feel, then select "Substance" from the Look and Feel dropdown. You will have the option to choose from these themes:

  • Autumn
  • BusinessBlackSteel
  • BusinessBlueSteel
  • Business
  • Cerulean
  • CremeCoffee

About Substance:

The Substance project has been resurrected by the original author, Kirill Grouchnikov. The latest release was on Oct 17. 2017. The version number of this plugin is a little odd, since it was following the forked Insubstantial project, which has died, but moved version numbers ahead. The next release from Kirill will be 8.0, so the version number of this plugin will track with that when it happens.

Changes

  • Version 7.4.7.1.00, updated to Substance 7.1:
    1. Removed dependency on laf-widget.jar and laf-plugin.jar as those are now incorporated directly into substance.jar.
  • Version 7.3.1, 28 Dec 2015, Bug fixes:
    1. Fixed null pointer exception in SubstanceColorSchemeUtilities.
    2. Adjusted class loading so theme properties are found on look and feel change.
  • Version 7.3, 31 Jul 2015, updated to Substance 7.3:
    1. Added new Cerulean theme.
    2. Added menu search feature.
  • Version 7.2, 14 Nov 2012, updated to Substance 7.2.
  • Version 6.1, 16 Nov 2010, initial release as jEdit plugin.
