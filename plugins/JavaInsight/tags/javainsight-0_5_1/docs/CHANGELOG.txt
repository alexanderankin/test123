- Disassemble or decompile with BCEL (André Kaplan)
- New Jasmin, Class and Jode virtual file systems (André Kaplan)

- Moved package buildtools to javainsight.buildtools because of name clashes

- Bugfix: dependency error in makefile.jmk (André Kaplan)
- Bugfix: Jode options (--pretty, --decrypt, --onetime) weren't used correctly;
  worked only correct every second time (Dirk Möbius)
