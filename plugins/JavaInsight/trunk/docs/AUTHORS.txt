The JavaInsight plugin uses the following external libraries:
- The Jode Decompiler, a program and library written by
  Jochen Hoenicke (jochen@gnu.org). Jode is available at:
  http://jode.sourceforge.net.
- BCEL (ByteCode Engineering Library) written by
  Markus Dahm (markus.dahm@berlin.de). BCEL is available at:
  http://jakarta.apache.org/bcel
- The Procyon Decompiler, a program and library written by
  Mike Strobel (mike.strobel@gmail.com). Procyon is available at:
  https://bitbucket.org/mstrobel/procyon/wiki/Java%20Decompiler
