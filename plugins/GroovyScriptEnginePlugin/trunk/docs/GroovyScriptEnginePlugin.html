Getting value from your scripts

The Groovy ScriptEngine supports returning values from your scripts in a few ways.

Using "return"

		  def myScriptValue = "Hello World";
		  return myScriptValue;
=> "Hello World"

The last object created

		  def myScriptValue = "Hello World";
		  def myScriptValue2 = "Goodbye World";
=> "Goodbye World"

The last evaluation

		  def myScriptValue = "Hello World";
		  def myScriptValue2 = "Goodbye World";
		  myScriptValue;
=> "Hello World"

Using the magic variable, "SCRIPTVALUE"

		  SCRIPTVALUE = "Hello World";
		  if(SCRIPTVALUE != "") {
			  SCRIPTVALUE += ", and nice to meet you";
		  }
		  def foo = "bar";
=> "Hello World, and nice to meet you"
SCRIPTVALUE will not work if defined using "def". For more information please refer to the Groovy documentation on "Scoping and the Semantics of def"

Using the SCRIPTVALUE variable is the only way to absolutely guarantee what will be returned. The ScriptEnginePlugin looks specifically for this variable before using the Groovy ScriptEngine's result.
