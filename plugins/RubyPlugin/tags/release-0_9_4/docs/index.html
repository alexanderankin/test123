Ruby Editor Plugin

Plugin features include integrated Ruby docs, syntax error highlighting, auto indent and insert end, structure browser, file structure popup, navigation shortcuts, and progressive selection.

You configure the plugin's keyboard bindings via the menu Utilities->Global Options->Shortcuts->Plugin: RubyPlugin.

Syntax error highlighting

Highlights syntax errors as you code.

Auto indent and insert end

You'll never have to type end again, after you bind the auto-indent action to the enter key!

File structure popup

Quickly navigate a file with type ahead method find via the file structure popup.

Structure browser window

View the file structure at leisure via the struture browser window.

Navigation shortcuts

Use the navigation shortcuts to jump around methods and syntax errors in a file.
