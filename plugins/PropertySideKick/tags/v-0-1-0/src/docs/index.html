Introduction

A SideKick for standard java property files. This may seem useless since property files are flat in the first place, but the real value comes in the parsing and error notification. Plus, it is really nice to have the properties in a sorted list -- open jEdit's property file, for a good example. There are no settings for this plugin, it is a pretty basic SideKick extension.
