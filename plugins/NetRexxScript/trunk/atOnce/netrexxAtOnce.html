NetRexx at Once

Quick Guide for Java Developers

This guide is about NetRexx: a language derived from Rexx and Java. This document is intended for Java Developers who don't know anything about NetRexx and want to start quickly with a list of differences from Java. For more information about this powerful language see the Official NetRexx Site. There you can find all the necessary tools and the complete documentation.
