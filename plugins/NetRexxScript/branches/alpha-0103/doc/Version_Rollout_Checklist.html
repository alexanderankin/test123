Version Rollout Checklist

  1. Are all help files uptodate?
  2. Are changes documented in the modules?
  3. Is the version number updated in the .props file?
  4. Is the version number updated in the build.xml file?
  5. Are the updated helpfiles and distributions uploaded to my site?
  6. Is all modified code committed and tagged on sourceforge?
  7. Is the release doc updated and the test procedure completed?
