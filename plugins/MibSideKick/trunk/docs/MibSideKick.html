Introduction

The MibSideKick is a plugin that will parse your mib file and shows your errors.

It is based on the GPL mib parser Mibble v2.9.2

Usage

The MibSideKick parser automatically parse your smi-mib file like any other SideKick parser.
To follow imports, MibSideKick includes iana and ietf mibs. It also adds in it's search path the folder of your buffer, and you can add custom search path in the option pane
