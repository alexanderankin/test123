Introduction

Checkstyle plugin integrates the checkstyle library (http://checkstyle.sourceforge.net) in jEdit. It allows you to check your java sources

Checkstyle Usage

It is possible to check the current buffer or check all open buffers from the menus.
You can also check a folder from the VFS browser.

Limitations

Checkstyle only checks saved files, so if a buffer is dirty it will be skipped. And it only checks java sources
