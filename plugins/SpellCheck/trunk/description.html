SpellCheck is an jEdit plugin to perform spell checking of text in a jEdit buffer. You will need to install Aspell (http://aspell.net/) before using this plugin. Aspell is available for Windows (installer available from http://ftp.gnu.org/gnu/aspell/w32) and a variety of UNIX platforms.

This plugin provides: an jEdit action to spell check selected text, configurable location of the Aspell executable, spell check dialog to allow the user to validate each suspect word (the dialog has buttons for: Cancel, Change, Change All, Ignore, and Ignore All), it shows the suspect word, lists suggested replacements, and has a field where the user can type in the replacement word.
