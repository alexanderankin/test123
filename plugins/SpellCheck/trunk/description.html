SpellCheck is an jEdit plugin to perform spell checking of text in a jEdit buffer.

This plugin provides: an jEdit action to spell check selected text, a spell check dialog to allow the user to validate each suspect word (the dialog has buttons for: Cancel, Change, Change All, Ignore, and Ignore All), it shows the suspect word, lists suggested replacements, and has a field where the user can type in the replacement word. It also features user-editable dictionaries.
