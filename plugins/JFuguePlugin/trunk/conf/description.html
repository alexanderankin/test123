A jEdit plugin to bring the JFugue API to jEdit users and provide some basic functionality to help users create music using the JFugue API. The Plugin provides functions for playing the entire buffer, or any selected text through Java's sound system, using your Java sound bank. These and two other functions are available in beanshell, where you can play any given music string.

Releases:
0.6.1 - Three new functions were added to expose JFugue's saving capabilities. You can now save selected text or the entire buffer contents to a midi file. The extra function to save any given music string is only available through beanshell.

0.5.2 - First release. Plugins contains 3 functions for playing selected text and buffer contents. The last function is to play any given music string and is only available through beanshell.
