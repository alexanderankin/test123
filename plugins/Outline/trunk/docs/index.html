Outline plugin

Provides a SideKick-like tree which displays folds rather than language-specific structures. Outline will work with any folding mode other than the SideKick folding mode. That would be a waste of time as it would just be duplicating the SideKick Structure Tree.

See the Wiki page for more info.

Bugs and feature requests can be submitted at http://community.jedit.org/?q=project/Outline.
