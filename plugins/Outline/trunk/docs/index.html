Outline plugin

Provides a SideKick parser service which displays folds rather than language specific structures.

You can switch between "outline" and other parsers from the SideKick parser combo box.

Outline will work equally well using sidekick or explicit folding mode. They should be the same.

Outline will work with other editing modes as well, although it requires that the {{{ fold symbols }}} are inside proper comments for the language of that editing mode.

See the Wiki page for more info.

Bugs and feature requests can be submitted at http://community.jedit.org/?q=project/Outline.
