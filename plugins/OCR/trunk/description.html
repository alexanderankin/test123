OCR documentation

Written by Shlomy Reinstein, June/July 2009.

Description

The OCR plugin provides an interface to the GOCR tool - an OCR (Optical Character Recognition) program, developed under the GNU Public License. The plugin provides actions to: - Extract text from a picture file (PBM format). - Take a screen capture and extract text from a selected rectangle in it. The extracted text is then imported into a new, untitled buffer.

Setup

  • Get the GOCR binary for your operating system. You can find binaries for download on the GOCR website. Set the path to the GOCR executable in the plugin option pane.
  • If you want to use the screen capture action, you need a converter from PNG image format to PPM image format, because the plugin saves the screen captures in PNG format, and GOCR is not capable of reading this format.
    One such converter is "gbmconv", which can be downloaded from the GBM website, but any other command-line converter can be used as well. The converter should take 2 arguments, the PNG file to convert, and the output PPM file to create.
    Set the path to the converter executable in the plugin option pane.

Tips...

I've found out that GOCR tends to produce bad results with small / normal fonts, but very good results with large fonts. If you get bad results using this plugin, try increasing the font size gradually until you get satisfying results. Sometimes you may find that you need to use VERY LARGE fonts for GOCR to work properly.
