JavaScriptSidekick 0.1

Requirements:
	* jdk 1.4.1
	* jEdit 4.2pre8
	* PerlSideKick 0.4 (requires SideKick 0.3 and ErrorList 1.3.1)

Announcement:
	release 0.1

Source:
	cvs.sourceforge.net: jedit/plugins/JavaScriptSidekick
	tag: release-0_1

Short Description:
	A simple sidekick parser for Javascript.

Long Description:
	This sidekick plugin parses Javascript source and analyzes it, using regular expressions (an excentric programming style will probably confuse the plugin).
	It is derived from classes in the PerlSideKick plugin.
