Introduction

The ShortcutSaver plugin provides twenty configurable actions so you can have different
key mappings for different edit modes.

You can configure it to invoke one action when you're in html edit mode, another
when you're in java mode, another for perl, etc. etc., all using the same shortcut.

You can even assign more than one action to an edit mode. A popup list will appear
for you to select which one to invoke, unless you check the action chaining box,
in which case, each action will be invoked in the order specified, allowing for simple
batch type processing.

When assigning actions, you will see an "edit mode" named "Default". Any action added
here will be invoked unless overidden by an edit-mode specific action.
