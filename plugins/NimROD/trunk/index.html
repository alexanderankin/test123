NimROD Look and Feel Plugin

Author: Dale Anson
Version: 1.2d, Nov 2020

This plugin provides the NimROD Look and Feel to jEdit. Note that the NimROD jar file included with this plugin has been slightly modified to accommodate the 48 bit colors provided by jEdit's color chooser.

Configuration

All configuration for this plugin is done in the Look and Feel plugin.
Go to Plugins - Plugin Options - Look And Feel, then select "NimROD" from the Look and Feel dropdown. You will have the option to configure a theme by clicking the 'NimROD Theme Editor' button.

The configuration dialog has three tabs, "Basic", "Advanced", and "Themes".

In the Basic tab, you can set two colors, and all remaining colors will be automatically calculated to fit nicely with those colors. You can save a theme by clicking the "Save" button. You can save the theme either as a update to the current theme or save it with a new name. Note that just clicking the "Ok" will apply the changes, but they won't be saved in the "Themes" list.

In the Advanced tab, you can set all colors individually, plus you can set values for Black (mostly used for text color), White (mostly used for text field backgrounds), and opacity for frame borders and menus. You can save a theme by clicking the "Save" button. You can save the theme either as a update to the current theme or save it with a new name. Note that just clicking the "Ok" will apply the changes, but they won't be saved in the "Themes" list.

In the Themes tab, you can select existing themes to use. There are 6 pre-defined themes:

  • Bluebird, a bluish theme
  • Canary, a yellowish theme
  • Flamingo, a pinkish theme
  • Osterich, a dark, brownish theme
  • Parrot, a greenish theme
  • Raven, a dark theme
  • Redwing, a dark theme with redish highlights
