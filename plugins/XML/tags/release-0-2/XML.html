XML Plugin By Slava Pestov
sp@gjt.org
The XML plugin performs on-the-fly validation of XML files. Errors are shown in the Plugins>Error List window, are are highlighted in the buffer.

The XML plugin also displays an element tree in the Plugins>XML Tree window. This window is floating by default, but can be set to dock into the current view in the Docking pane of the Utilities>Global Options dialog box.

Clicking on an element in the tree will move the caret to its position; moving the caret using any other means will automatically expand the tree and select the appropriate node.
