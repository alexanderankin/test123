Markdown Sidekick

A simple sidekick for displaying Markdown files.

Options are minimal, headers are always shown, visibility of the following may be turned on or off in the plugin options:

  • Show Paragraphs - if checked, the first line of each paragraph will be displayed.
  • Show Block Quotes - if checked, the first line of each block quote section will be displayed.
  • Show Code Blocks - if checked, the first line of each code block will be displayed.

Parsing is based on the syntax guide found here: Markdown: Syntax
