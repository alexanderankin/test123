ClearCase plugin.

Tested on ClearCase v3.2.1, v4.1 on Windows NT v4.0

Features: Interactive ClearTool shell in the jEdit console. The following commands are available from the Plugin menu: uncheckout, checkout, checkin, mkelem, diff -pred, lshistory, lvstree, update, cleardetails, clearmrgman

Installation: Put the jar in the jEdit jars folder.

Goto Global Options->ClearCase to adjust path to cleartool executable.

 

Tips

Keyboard and toolbar shortcuts can be added from the "Global Options" menu.

Goto Global Options->ClearCase to tune file reload delay time. Some delay time must be present to allow ClearCase to update the VOB. This time can vary from system to system. Default value is three (3) seconds.
