PythonShell

Damien Radtke

Introduction

The PythonShell for jEdit offers a command line shell from the Console Plugin for the Python language.

It also offers actions for you to evaluate selections, buffers or files in the shell.

It runs the command-line tool python (or ipython) and requires it to be in your path, or else you can set the location of the executable from plugin settings - PythonShell - Python executable.

Enjoy!
