Expand an abbreviation

To expand an abbreviation, you have to be in the mode the abbreviation is associated with, and the cursor should be placed after the abbreviation. Then press the TAB key, the template will be inserted and the first variable will be selected. If you change the text of the variable it changes everywhere the variable is placed in the template. By pressing the TAB key once more, the next variable is selected. If you want to jump back to a earlier variable you should just press S+TAB.
