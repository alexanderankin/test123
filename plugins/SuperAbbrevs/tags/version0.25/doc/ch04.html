Chapter 4. Template generation

This is only for advanced users.

It's possible to generate a template with beanshell code, you embed code into the template in between <# code #>, the template is then turned inside out, so all the normal text is turn into print statements, and the code will be executed. It's possible to print new code to the template with <#= code #> where the code have to evaluate to a string. All the functions in the files .jedit/SuperAbbrevs/template_generation_functions.bsh .jedit/SuperAbbrevs/abbrevs_functions.bsh is available, as well as a variable selection, holding the selection.

This example shows how to generate a HTML multiplying table of 5 rows and 5 columns, not that usefull but only imagination hold you back. 

<table>
<# for(i=0; i<=5; i++){ #>
  <tr>
  <# for(j=0; j<=5; j++){ #>
    <td><#= i*j #></td>
  <# } #>
  </tr>
<# } #>
</table>
