SuperAbbrevs

Author: Sune Simonsen

This will soon be updated

This plugin gives you eclipse style code templates. You can define a template wich contains variables. You assosiate the template with a abbreviation, if you have the caret placed after the abbreviation and execute the command:

Plugins-> SuperAbbrevs -> Expand Abbreviation

it will be expanded. You can then use the SHIFT-TAB and TAB keys, to jump back and forth between the variables.

Template syntax

This is best explained with an example.

	Iterator ${2:iter} = ${1:list}.iterator();
	while ($2.hasNext()){
		${3:Type} ${4:element} = ($3)$2.next();
		$end
	}

The above example, shows a java template, with the variables $1, $2, $3 and $4. Variable $1's default value is list, and $2's default value is iter, and so on. When this template is inserted the following code will be inserted, and the list will be selected. 

	Iterator iter = list.iterator();
	while (iter.hasNext()){
		Type element = (Type)iter.next();
		
	}

There are to types of variables, a variable that defines the default value of the variable, and a plain variable. Each has a tab index, if the variable should have a default value, it should be define in the first occurrence of the variable. Like index 2 has defined it's default value to iter (${2:iter}). When you have defined the default value you can just use the variable index $2 to referrer to the variable. The template also has to contain a $end variable, that represent the last entry, if you write in the $end variable the template will be disabled and you go back to normal mode.

I suggest that you bind Expand abbreviation to a key.
