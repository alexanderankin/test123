Ancestor Version 1.1.1 (Apr 24, 2014)
Matthieu Casanova <chocolat.mou@gmail.com> 
Introduction

This plugin is a toolbar that show you the full path of your file with every ancestor folders as button that open the file system browser on them

History

  • 1.1.1 (April 24, 2014)
    • Do not show toolbar on plain view
  • 1.1.0 (December 15, 2011)
    • The parent folder of the current buffer is now bold
  • 1.0.6 (January 4, 2011)
    • Pressing control+click on Ancestor buttons will close the opened files of all subfolders (Thomas Katzlberger)
  • 1.0.5 (September 9, 2010)
    • The margin around the toolbar button is removed for a most compact look
    • The toolbar was ignoring the VFS change of parent paths (FileRootsVFS usually is path of everything)
  • 1.0.4 (November 25, 2008)
    • Fixed bug : If the file was renamed the bar was not updated (Patch #2343308 from Nick Clarke)
  • 1.0.3 (September 22, 2008)
    • Fixed a NPE that happened when creating a new view
