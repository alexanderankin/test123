ClojureShell maintains a running Clojure environment and Read-Eval-Print-Loop in the Console.

Actions are provided to quickly evaluate selected text, an open buffer, or a chosen external file in the shell, making it easy to run quick tests before committing code to your project.

The Read-Eval-Print-Loop (REPL) is hosted by default by the clojure.jar installed using the Clojure plugin. ClojureShell can instead be configured to use the Leiningen tool. In this scenario, if the ProjectViewer plugin is also installed, the Leiningen REPL will be invoked in the project root directory, using the project's project.clj file.

Source: https://sourceforge.net/projects/jedit-plugins/files/ClojureShell
