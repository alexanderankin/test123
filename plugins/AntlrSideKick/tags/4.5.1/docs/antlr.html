Antlr SideKick Plugin

Dale Anson

ANTLR (ANother Tool for Language Recognition) is a powerful parser generator for reading, processing, executing, or translating structured text or binary files. It's widely used to build languages, tools, and frameworks. From a grammar, ANTLR generates a parser that can build and walk parse trees.

This plugin provides a SideKick to make working with ANTLR grammar files easier.

There is one action provided, "Generate files". If the current buffer contains an Antlr 4 grammar file, executing this action will generate the Antlr lexer, parser, etc, files into the same directory.
