RFCReader

By Matthieu Casanova

RFCReader provides two fold handlers, and a hyperlink source.

Fold handlers

When editing files in rfc edit mode, you can have folds that are aware of pages, or sections and subsections of the RFC documents. You can change the default fold handler for an edit mode from Global Options - Editing and select your mode, un-check "use default settings" and then select the desired fold mode.

Fold handlers, as always, can be used as Section headings in the SideKick structure browser, by using the Outline plugin.
