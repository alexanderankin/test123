Introduction

RFCPlugin Usage

The purpose of RFCPlugin is to make RFC reading more convenient

To do that the plugin provide two folding modes, one per chapters, one per pages. This is very useful when combined with Outline plugin that will show you all folds in the Sidekick's tree (if you use Chapters folding, you see all chapters in the tree)

It provides also some hyperlinks, when you press control and move you cursor on the text, it will recognize references to others RFC and open them if you click. And also will be able to browse directly the table of contents

The plugin also have a convenient way to open RFCs directly from Internet by searching their numbers, or any word in their titles. It is also possible to use a RFC index plugin that will contains the index of the content of the RFCs. To do that, install the RFC index plugin you want and choose it in the RFCReader options
