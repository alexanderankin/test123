Introduction

In the beginning of 2003 I found the phpeclipse project, a plugin to do php in Eclipse. I worked on it, and finaly wrote an alternative php parser for the project. The matter is that I don't like SWT very much so when I discovered discovered jEdit I loved it. So I decided to write a plugin using my php parser. It uses ErrorList plugin to show the errors.
This plugin uses some classes from phpeclipse. It is not a complete replacement of phpeclipse, that will probably always remains more complete. You can find more infos at http://www.phpeclipse.de

Usage

There are no actions to do the files finishing by .php will be parsed
