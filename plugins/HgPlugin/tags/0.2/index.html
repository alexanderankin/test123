Introduction

The HgPlugin for jEdit is the place for Mercurial-related services in jEdit. Currently, the only thing it provides is a ProjectViewer file importer. This means you can choose ".hg/dirstate" when you are importing files in ProjectViewer.

It opens the dirstate file as a local file and uses that as the list of files to import.

If you want to develop something more substantial in terms of integration with jEdit, feel free to join this project and collaborate.

Enjoy!
