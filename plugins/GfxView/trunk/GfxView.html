GfxView plugin

Author and facts

This jEdit plugin is coded by Ludovic ANCIAUX (Ludovic.Anciaux@advalvas.be) and based on QuickNotePad "learning principles" plugin by Slava Pestov (http://www.jedit.org). It is free for any non commercial use but ImageroReader library has its own licence you also have to have read before using this software.

Purpose : a basic viewing picture area

There are a plugin menu, a FileSystemBrowser plugin menu and two panels :

  • Plugin menu functions are showing history panel, getting next and previous picture from history and showing a remote picture file from internet/intranet.
  • One can display a file in FileSystemBrowser (through VFS plugin menu)
  • An history panel that shows history list of latest additions (20 by default).
  • A display panel that shows picture choosen from history panel or dialog boxes. This panel has 3 buttons for zooming, unzooming and restoring original size. User has the possibility to draw a zoom window while maintening CTRL key pressed. While Ctrl+Click increases zoom level, Ctrl+Shift+Click decreases it.

Limitations

  • It shows only files formats recognized by ImageroReader library (JPG/PNG/TIF/BMP). Previously it only supported Java JRE native formats (PNG/JPG).
  • Loading files from HTTP queries is rather basic
