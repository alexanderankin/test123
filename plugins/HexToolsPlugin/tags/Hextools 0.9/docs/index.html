Presentation

HexTools Plugin extracts data from a hex dump and saves into a file or into the clipboard.

Operation

Say you have a hex dump like this in a jEdit buffer and you'd like to save the data as binary file (e.g. because your favorite ORB only saves message data to the log). 
00000000  6a 45 64 69 74 20 69 73  20 61 20 67 72 65 61 74  |jEdit is a great|
00000010  20 74 65 78 74 20 65 64  69 74 6f 72 2e 0d 0a     | text editor...|
0000001f
Select the data area of the hex dump (it is recommendable to use rectangular selection) and choose Plugins->HexTools. You get this dialog, where you can choose to save the hexdump data or copy it as String to the clipboard using the selected encoding. The conversation works by stripping all whitespace from the selection and parsing two hex digits at a time.
