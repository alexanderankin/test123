MetalColor plugin

Author: Jocelyn Turcotte

Introduction

With the MetalColor plugin you can customize the colors of the default Metal look and feel.

Usage

First make sure that you have selected the Metal option from "Global Options / Appearance / Swing Look & Feel" and that the Look And Feel plugin is set to None.

The plugin only consists of an option pane that allows you to choose 3 base colors:
  • Control color, changes controls color like buttons, scroll bars and selected menu
  • Background color, change background elements color like panels and menus
  • Text color, which modifies UI controls text color (not in jEdit text area)
Just change these colors, press apply to preview then restart jEdit to correct small glitches, voilà!
