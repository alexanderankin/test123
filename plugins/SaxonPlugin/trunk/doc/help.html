Saxon plugin

About

SaxonPlugin bundles the Saxon XSLT and XQuery Processor from Saxonica Limited Home Edition, version 9.2 or later. Saxon is available at http://www.saxonica.com/.

The Source Code version of the Covered Code (that is, the source code of Saxon-HE) is available under the terms of the Mozilla Public License version 1.0, and may be obtained from the Downloads directory for the Saxon project on SourceForge, at http://sourceforge.net/projects/saxon/files/Saxon-HE/9.4/. The precise version of the source archive for a particular Saxon maintenance release is the same as the SaxonPlugin version. For instance, source archive for Saxon-HE 9.4.0.6 is available here: http://sourceforge.net/projects/saxon/files/Saxon-HE/9.4/saxon9-4-0-6source.zip/download

Support for packaged stylesheets has been introduced in Saxon 9.4. Quoting Saxon's documentation (retr. 2012-10-30):

A new facility is available in Saxon 9.4 allowing an XSLT stylesheet to be distributed to users in a format such that the source code of the stylesheet is not readily visible.
Running packaged stylesheets is enabled via the saxon9-unpack.jar library, part of the Saxon-HE distribution. We distribute it alongside the main saxon9he.jar library.

Documentation for Saxon is available from saxonica.com.
