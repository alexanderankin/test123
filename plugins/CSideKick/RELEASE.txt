CSidekick 0.1 (not released or ready yet)

Requirements:
	* jdk 1.5
	* jEdit 4.3pre5
	* SideKick 0.4
	* ProjectViewer 2.1    
	* ErrorList 1.4

Source:
	http://svn.sourceforge.net/viewcvs.cgi/jedit/plugins/CSideKick/

Short Description:
	A simple sidekick parser for C/C++ that uses the JavaCC grammar for C and C++,
    and also combines the header and the C++ into a single sidekick view.
	Most of the code is based on JavaSideKick, adapted for C++ where appropriate.
