Event Handling.

As of ProjectViewer 3.0, events are sent using jEdit's EditBus. There are three kinds of event:

  • {@link projectviewer.event.ProjectUpdate}: project-specific events, which deal with changes in the project's internal data
  • {@link projectviewer.event.ViewerUpdate}: GUI updates, which deal with changes in the plugin's user interface.
  • {@link projectviewer.event.StructureUpdate}: changes in the organization of projects and groups managed by the plugin.
