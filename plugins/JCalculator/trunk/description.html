Long Description: This plugin 'JCalculator' can evaluate expressions you write in the dockable or in the text area. It supports the following features:

  • Bitwise operations such as the operators left/right shift, and, or etc.
  • Functions (till now there is only the function 'POWER' has been appended).
  • Evaluate selection as formula from jEdit.
  • Shows record of calculations and results.
  • Highlights input errors.
