Rename Plugin

Brad Mace

This plugin allows you to rename plugins, actions, dockables and option panes in jEdit. This helps you if you find yourself short on screen space, or wish to translate words into your native language to help you remember them better.

There are no actions or dockables for this plugin. Everything that can be done with this plugin is exposed from the Plugin Options - Rename option pane.

One interesting side-effect of this plugin is that it makes permanent changes to the labels in the user's properties file. Even after uninstalling the plugin, the renamed things remain.
