The Menu Editor Plugin Written by Shlomy Reinstein, August 2010. The Menu Editor plugin allows you to rearrange the jEdit menu (File/Edit/Search...), shown at the top of each view. You can rearrange the items in each menu, remove items from a menu, or add items to a menu. In addition, the plugin allows you to rearrange the menu bar itself - reorder the menus, add new menus or remove existing menus.
