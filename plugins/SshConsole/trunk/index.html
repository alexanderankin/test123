SSH Console Plugin

Alan Ezust

$Date$

This is the SSH Console plugin. Its user interface is fairly minimal, because it attempts to reuse as much of Console and FTP plugins as possible.

Instructions for use:

  1. Navigate your File System Browser to a remote host via sftp://.
  2. If this is your first time connecting, Enter your user/password/key/passphrase there.
  3. Open the Console, select the "ssh" shell.
  4. Start typing.

Click to change directories

A nice feature of this plugin is that you can change the working directory of the remote shell session by clicking on files in the File System Browser. Note, that this will only work if you have Plugin Options - Console - General - On VFS Node Select, cd to node dir option switched on.

Enjoy!
