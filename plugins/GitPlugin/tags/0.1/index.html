Introduction

The GitPlugin for jEdit is the place for git-related services in jEdit. Currently, the only thing it provides is a ProjectViewer file importer. This means you can choose "git ls-files" when you are importing files in ProjectViewer.

It runs the command-line tool and requires the tool to be in your path, or else you can set the location of the executable from plugin settings - git - git executable.

If you want to develop something more substantial in terms of integration with jEdit, feel free to join this project and collaborate.

Enjoy!
