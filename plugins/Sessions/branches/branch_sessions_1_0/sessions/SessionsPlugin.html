Introduction

The Sessions Plugin maintains lists of open files - called sessions (aka workspaces) - and offers some functions to switch between them.

Key Actions

The Sessions Plugin is fully keyboard navigable.

A prerequisite is that you assigned a keyboard shortcut to the action Manage Sessions.... You can assign a keyboard shortcut to this action under Utilities>Global options>Shortcuts. I suggest "A+s".

Key Shortcut Action
[Alt-s] Show Session Manager
[Up] / [Down] Choose session
[Enter] / [Return] Jump to/Open selected session
[Esc] Close dialog

Customization

Appearance

You can customize the Sessions Plugin settings in Utilities>Global options>Plugin Options>Sessions. The following options are available:

  • Autosave Sessions
  • Close all files before switching to new session
  • Show Session Switcher toolbar
    • Show below JEdit's main toolbar
    • Show inside JEdit's main toolbar (at the end)
    • Show inside BufferList
      This requires BufferList 0.8 or higher.
      The toolbar can only be shown, if the BufferList is docked (ie. non-floating).
    • Show "Session:" label
