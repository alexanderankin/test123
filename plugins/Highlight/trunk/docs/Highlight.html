Introduction

This plugin is a very simple plugin that will highlight a word you selected. I found this in Intellij Idea and loved this feature so I added it here.

Highlight Usage

There are only two actions for the moment : highlight and cancel highlight. The first one will highlight the word you selected, or select the word pointed by your carret if no text was selected, and highlight it everywhere in your text. The second action will simply remove the highlighting.
