Quickstart

The one-minute quickstart to using CamelComplete is to
  1. Install CamelComplete from JEdit’s Plugin Manager
  2. Ensure that Simple Mode is selected in the plugin’s options
  3. Bind CamelComplete->Simple Complete to an easy shortcut from JEdit’s general preferences. This configuration will work for languages like C and Java, that use either “CamelCase” notation, or underscore separators. To have CamelComplete work with other types of identifiers, you’ll need to delve into Advanced Mode, which is detailed in the manual.
