Introduction

This plugin provides an interface within jEdit to the jEdit Community website's contributed macro archives.

Usage

Browsing Available Macros

Invoke Macro Manager from the Plugins menu. This will bring up a list of all available macros. Use the tree list exactly as you would jEdit's Plugin Manager. The macro list is quite large, and is therefore cached to reduce the time spent downloading. To get the latest list, click the refresh button.

You can use the Sort button to toggle between listing the most recently added macros and an alphabetical list. Toggling the sort, however, will deselect any selected macros.

You can use the search field to restrict the listing to those macros whose descriptions or names contain, in any order, each search term.
