Introduction

This plugin provides an interface within jEdit to the jEdit Community website's contributed macro archives.

Usage

Browsing Available Macros

Invoke Macro Manager from the Plugins menu. This will bring up a list of all available macros. Use it exactly as you would jEdit's Plugin Manager.
