Introduction

The Qt4Jedit plugin is the home for Qt4 integration features with jEdit. Currently, the only actions it provides are related to the new Qt 4.4 remote controlled assistant help browser.

The actions provided are:

  • Start assistant
  • Stop assistant
  • Activate Identifier - If there is a known help page for the identifier under the cursor, open that page.
  • Activate Keyword - Perform keyword search in the index, and activate the first exact match.
  • set url - Open selected URL in assistant

All of these actions work on the selection, or the word under the cursor.

Enjoy!
