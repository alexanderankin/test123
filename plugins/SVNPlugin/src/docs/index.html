Subversion Plugin

Author: Dale Anson

First attempt at a Subversion plugin. Access the commands via the context menu in ProjectViewer. Set/view repository location in ProjectViewer Properties. Set username and password (if required) for checkout in ProjectViewer Properties.

This plugin complies with Subversion 1.4. If you use an older Subversion client that is pre-1.4, you may have some problems. Subversion changed the format of the files stored in the .svn directories with 1.4. All 1.4 compliant clients will automatically and silently convert the format of those files, which will render them unreadable by older clients. If you must continue to use your older client and are unable to upgrade it to be Subversion 1.4 compatible, you should not use this plugin.

To check out a project that already exists in subversion:
  1. Create the project in ProjectViewer
  2. Right click on the project node in ProjectViewer
  3. Select "Properties"
  4. Select "Subversion Settings"
  5. Enter the repository URL, username, and password. Leave username and password blank if the repository allows anonymous access. Don't forget to add "/trunk" to the end of the URL, unless you want all the branch and tag info locally.
  6. Click the "OK" button.
  7. Right click on the project node in ProjectViewer
  8. Select "Subversion"
  9. Select "Checkout"
  10. Double check the entries on the dialog
  11. Click the "OK" button
  12. Wait for the checkout to complete
  13. Right click on the project node in ProjectViewer
  14. Select "Re-import files"
  15. Select "Use CVS or SVN settings"
  16. Click the "OK" button.
