Summary

ClangCompletion provides code completion and syntax checking for c, c++ and objective-c using the clang and llvm.

Usage

ClangCompletion needs clang inside the system path. Please download and install llvm from http://llvm.org. It also provides an option pane inside project viewer to config including pathes and precompiled definitions. You need to fill in the same options as in your makefile or your IDE.

ClangCompletion will check the syntax every time you save your c, c++ and objc file. It will put the errors into errorlist. It provides completion function via Completion Plugin. The completion will automatically show or need a key to popup depends on the completion plugin options.
