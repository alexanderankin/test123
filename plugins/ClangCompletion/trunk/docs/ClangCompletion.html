Summary

ClangCompletion provides code completion and syntax checking for c, c++ and objective-c using the clang and llvm.

Usage

ClangCompletion needs clang inside the system path. Please download and install llvm from http://llvm.org. It also provides an option pane inside project viewer to configure include paths and precompiled definitions. You need to fill in the same options as in your makefile or your IDE.

ClangCompletion will check the syntax every time you save your c, c++ and objc file. It will show errors in the ErrorList. It provides completion function via Completion Plugin. The completion will automatically show or need a key to popup depending on the Completion Plugin options.
