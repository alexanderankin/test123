Java Compiler (Kevin A. Burton)

Java Compiler basically has 3 options:

  • Build File

    Builds the file currently being edited.


  • Build Package(s)

    Builds the package currently being edited. This is determined by finding out the package that the current file is in and then goes it's root directory and then finds all sub packages. It does a smart compile in that if you have compiled a file and it's .java file hasn't changed then it doesn't recompile it.


  • Rebuild Package(s)

    Does basically what "Build Package(s)" does but always compiles every file.
