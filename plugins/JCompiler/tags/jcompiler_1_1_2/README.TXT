JCompiler - A java compiler plugin for jEdit
============================================

INFORMATION FOR DEVELOPERS

If you want to compile the sources yourself: The files
	jcompiler/NoExitSecurityManager.java  and
	jcompiler/NoExitSecurityManager2.java
can only be compiled with a Java2 javac, _not_ a JDK 1.1 javac.

The compiled code run will run on old JDK's nevertheless.
