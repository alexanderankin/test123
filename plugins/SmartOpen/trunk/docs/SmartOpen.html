Introduction

SmartOpen plugin is made to help you opening your files quickly. Like others (FastOpen, OpenIt plugins), you will be able to type a portion of the name of your file and you'll get a list of matching names

Usage

Setup

To use SmartOpen you have to choose which directories will be indexed in the options. Or use ProjectViewer and SmartOpen will automatically index files of the current project.

Usage

SmartOpen can be used in the Toolbar, or as a floating window. You can search for any file that is indexed, and also use CamelCase :
For example to search for a file named AbstractOptionPane.java the camelcase are AOP. You can search your file with any combination using those capitals chars :

  • AOP
  • AOPane
  • AbsOPane
  • AbsOptPane

and any other combination
