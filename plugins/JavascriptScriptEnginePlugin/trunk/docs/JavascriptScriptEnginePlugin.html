Getting value from your scripts

The Javascript ScriptEngine supports returning values from your scripts via the last evaluation.

The last evaluation

		  var myScriptValue = "Hello World";
		  var myScriptValue2 = "Goodbye World";
		  myScriptValue;
=> "Hello World" 
	     function printz(text) {
	        return text;
	     }
	     printz("Doh!");
	       
	       => Doh!
	    

	    
Using the magic variable, "SCRIPTVALUE"

	       
		  var SCRIPTVALUE = "Hello World";
		  if(SCRIPTVALUE != "") {
			  SCRIPTVALUE += ", and nice to meet you";
		  }
		  var foo = "bar";
	       

	       => "Hello World, and nice to meet you"
	       

	       For more information please refer to the "Rhino documentation"
	    

	    
Using the SCRIPTVALUE variable is the only way to absolutely guarantee what will be returned. The ScriptEnginePlugin looks specifically for this variable before using the Javascript ScriptEngine's result.
