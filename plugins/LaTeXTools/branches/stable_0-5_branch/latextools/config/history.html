Release Notes

LaTeXTools 0.5

Feature release:
  • Now uses the jEdit 4.2 API
  • Improved BibTeX and Label Navigators now take into account the file import structure. They also display more information in a sortable table.
  • Compilation commands integrate with Console plugin.
  • Image browser to look at the images in your document.
  • Various navigability improvements and utility commands.
  • Extended default.nav (tables, lists, verbatims)
  • New icon types : table, list, verbatim

LaTeXTools 0.2.1 (Jun 26 2003)

Bugfix release:
  • Problem with BibTeX Pane inserting multiple copies of the same citation fixed.
  • Now reparses the BibTeX and Cross-reference Panes are now updated on open, close and save.

LaTeXTools 0.2.0 (Apr 25 2003)

Implemented SideKick API - LaTeX document structure displayed in the SideKick Structure Browser. customizable multiple parsers are available as before, and are accessed via a separate (dockable) pane. A default parser is displayed if the Navigation pane is not required. BibTeX and Cross-reference dockable panes are still available. Some macros are also built in, and are accessible via the menu.

LaTeX Tools 0.1.1 (Nov 14 2002)

Bug Fix release. Navigator now performs properly when docked or floating.
Navigation files now parse the strings as regex's. Capturing groups are displayed as a concatenated string in the tree by default, or else the whole line is displayed if no groups are present.

LaTeX Tools 0.1.0 (Nov 9 2002)

Initial Release.

Features:

  • BibTeX citation completion

  • Reference completion

  • LaTeX Navigator

Known Problems

Navigator

The scrollpane displaying the tree in the Navigator does not resize properly with the dialog that contains it. The scrollpane does not display properly at all when docked.
