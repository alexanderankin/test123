Release Notes

LaTeXTools 0.5.9

Bugfixes:

  • [Bug 3410449] Fixed a deadlock that sometimes occured when switching from a latex buffer.
  • Fixed an NPE that occurred when re-opening jEdit on Mac OS X.
  • Fixed an NPE that occurred when closing a view.
  • Added support for the "part" document division.
  • Improved end-of-section detection.

LaTeXTools 0.5.8 (Sep 28 2010)

Bugfixes:

  • Error style switch - previously this would always be reset to the default value when loading the plugin under Linux. The "on first run" initialization is now recorded to avoid overwriting changes to this property by the user.

LaTeXTools 0.5.7 (Oct 22 2006)

Bugfixes:

  • BibTeX Navigator - when the entries were sorted by anything else then "Ref" and you wanted a reference to be inserted, the reference on the same position when sorted by Ref was inserted instead of the correct one - now the correct refference is inserted.
  • BibTeX Navigator - when sorting, ignore trailing whitespaces.

New features:

  • You can change the sort order of bibTeX entries by clicking the column header again.

LaTeXTools 0.5.6 (Jun 10 2006)

Requires jEdit 4.2final or 4.3pre3, Console 4.2.x, JDK 1.5.

  • [Bug 1488309] Fixed handling \(chapter|section)[short title]{long title}.
  • Made the LatexTools NavigationGroup combo box value persistent in properties, and also added it to the OptionPane.
  • Redesigned option panes.

LaTeXTools 0.5.2 (Jun 8 2005)

Bugfix release:
  • [Bug 1396913] Now works also for files (buffers) in the edit mode 'latex'.
  • [Bug 1112059] Added the option Latex error style switch to the option pane + using -file-line-error-style by default on Linux.
  • [Bug 1396942] build.xml: added some default values etc.

LaTeXTools 0.5

Feature release:
  • Now uses the jEdit 4.2 API
  • Improved BibTeX and Label Navigators now take into account the file import structure. They also display more information in a sortable table.
  • Compilation commands integrate with Console plugin.
  • Image browser to look at the images in your document.
  • Various navigability improvements and utility commands.
  • Extended default.nav (tables, lists, verbatims)
  • New icon types : table, list, verbatim

LaTeXTools 0.2.1 (Jun 26 2003)

Bugfix release:
  • Problem with BibTeX Pane inserting multiple copies of the same citation fixed.
  • Now reparses the BibTeX and Cross-reference Panes are now updated on open, close and save.

LaTeXTools 0.2.0 (Apr 25 2003)

Implemented SideKick API - LaTeX document structure displayed in the SideKick Structure Browser. customizable multiple parsers are available as before, and are accessed via a separate (dockable) pane. A default parser is displayed if the Navigation pane is not required. BibTeX and Cross-reference dockable panes are still available. Some macros are also built in, and are accessible via the menu.

LaTeX Tools 0.1.1 (Nov 14 2002)

Bug Fix release. Navigator now performs properly when docked or floating.
Navigation files now parse the strings as regex's. Capturing groups are displayed as a concatenated string in the tree by default, or else the whole line is displayed if no groups are present.

LaTeX Tools 0.1.0 (Nov 9 2002)

Initial Release.

Features:

  • BibTeX citation completion

  • Reference completion

  • LaTeX Navigator

Known Problems

Navigator

The scrollpane displaying the tree in the Navigator does not resize properly with the dialog that contains it. The scrollpane does not display properly at all when docked.
