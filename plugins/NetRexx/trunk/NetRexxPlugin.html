Introduction

The NetRexx plugin adds support to jEdit for interpreting or compiling and running scripts, macros, and programs written in NetRexx. Because the NetRexx language can be compiled into Java classes or directly interpreted, there are two modes available in this plugin - one for compiling and running NetRexx programs and another for interpreting scripts and macros.

Requirements

The minimum requirements are:

  • JRE 1.5 (tested with JRE 1.6.0_35)
  • jEdit 4.3
  • Windows or Linux (Mac status unknown)
  • NetRexxC.jar (Rexx Language Association Open NetRexx version 3.02 or later)
Additional plugins needed to activate all features (install before activating NetRexx plugin):
  • ErrorList plugin (for error highlighting)
  • CommonControls (for classpath builder function)
  • SideKick (for code structure browser)
  • The Console plugin for jEdit is required to use the Android IDE macros.
The ProjectViewer plugin is recommended for managing NetRexx projects and the SVNplugin is recommended for accessing NetRexx source repositories.

Note that this release may work with other Java and jEdit versions or other operating systems but these are the only tested environments at this time. I will adjust these requirements if anyone tests the plugin under different environments and reports success or failure there.

Installation

The 1.0.0 version is available from the jEdit Plugin Central site via the jEdit plugin manager.
Start jEdit, from menu select "Plugins"=>"Plugin Manager", select "Install" tab, check box for NetRexx Plugin, click "Install button".

Skip to step 4 below if installing via Plugin Manager.

You can also install it manually if needed:
  1. Delete any old plugin directories from previous versions before running the new plugin:  SettingsDir/NetRexx and SettingsDir/macros/NetRexx*
    Go to Utilities - Settings Directory to see where the jedit user settings directory is on your system.
  2. Download the NetRexxPlugin.jar file and the NetRexxC.jar file:
    https://kenai.com/projects/netrexx-plus/downloads/download/jEdit%20NetRexxPlugin/NetRexxPlugin.jar
    http://netrexx.org/netrexx/netrexxc/trunk/lib/NetRexxC.jar

  3. Copy both files to the user account's jEdit SettingsDir/jars directory.
  4. Start jEdit, then open the NetRexx dockable from the jEdit menu: "Plugins"=>"NetRexx"=>"NetRexx", then click the down arrow at upper left of window and select to dock (Top or Bottom work best).

  5. Verify installation: In a new netrexx buffer, enter the following line:
    say "Hello World"
    Click the Run button in the NetRexx window (right arrowhead). This output should appear in the output window: Hello World
