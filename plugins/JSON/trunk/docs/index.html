JSON Plugin

Dale Anson

This plugin provides some tools for working with JSON files:

  • A SideKick for browsing JSON files.
  • A beautifier to make JSON files human-readable.
  • A minifier to make JSON files machine-readable.

Many JSON files are machine generated in a single line, the beautifier arranges the file in a more human-readable tree-like format, and the SideKick will make browsing that tree easier.

In addition to beautifying a JSON file, this plugin has an action to "minify" a JSON file, that is, convert the JSON text into a single line minus whitespace, similar to many JSON files that are machine generated. Note that the JSON specification does NOT allow JS style comments (// and /* ... */), the minifier will strip any such comments. Further, the beautifier will fail if such comments are in a given JSON file.
