Jump works in conjunction with the ProjectViewer plugin and "Exuberant Ctags". It provides several actions which will accelerate your work with jEdit.
  • "Jump to File" - Show list of all files in current project.
  • "Jump to Tag" - Show list of all tags in current buffer.
  • "Search Tag" - Show list of files which contain the tag under the caret.
  • "Type a tag..." - Displays an input dialog for entering a tag to search.
  • "Jump Back" - Stack history of visited tags.
  • "Jump to Fold" - Shows list of folds in current buffer.
