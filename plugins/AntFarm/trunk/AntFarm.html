Overview

AntFarm allows you to execute builds using Apache's Ant utility from within jEdit. Currently, AntFarm supports compilation with javac and jikes, and will parse the output of both, displaying any errors in ErrorList.

AntFarm will further coorperate with ProjectViewer by reading a build.xml from the project directory of ProjectViewer's currently selected project, provided that one exists.

Keep in mind this is an early release and will probably have bugs in it. Also, it is not multi threaded and thus blocks jEdit until the build it is complete.

AntFarm requires:

  • JDK 1.1
  • jEdit 3.2pre10
  • EditBus Plugin
  • ErrorList Plugin
  • XML Plugin

Usage

Launch AntFarm using the Plugins menu item. Once open, you can specify your build.xml file by clicking on the browse button. Alternatively, you can type in the path directly. Alternatively, you can use ProjectViewer to administer projects. AntFarm will check the project directory of the currently selected ProjectViewer project for a build.xml. The string representation specifying the path will be colored red as long as your typing does not point to a valid file. After you have specified a valid build file, this file will be automatically parsed and the target combo box on the right side of the AntFarm pluging will be filled with the existing targets. In order to initiate a build, choose a target from the list and click the build button.

Notes:

  1. The build file text field makes use of JEdit's history text field so that all the build files that you type in will be remembered across editing sessions.
  2. It is possible that AntFarm will fail to parse some messages from javac and jikes. If this is the case, please submit the string in a bug report so that AntFarm can be extended to sucessfully parse it.

AntFarm uses the following libraries:

    A utility created by the Apache Software Foundation. It's home page can be found at http://jakarta.apache.org/ant/index.html.
  • Xerces
    An XML parsing library made available by xml.apache.org. This library is included with the XML plugin version 0.3 and later, which must be installed in order for AntFarm to function.
