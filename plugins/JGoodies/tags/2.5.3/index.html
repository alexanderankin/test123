JGoodies Look and Feel Plugin

Author: Dale Anson
Version: 2.5.3, Dec 2020

This plugin provides four JGoodies Look and Feels to jEdit:

  • JGoodies Plastic
  • JGoodies Plastic3D
  • JGoodies PlasticXP
  • JGoodies Windows (only if the current operating system is a version of Windows)

These look and feels were developed to help make java applications blend in on Windows. They look pretty good on other operating systems also.

Configuration

All configuration for this plugin is done in the Look and Feel plugin.
Go to Plugins - Plugin Options - Look And Feel, then select one of the "JGoodies" choices from the Look and Feel list. You will have the option to select a theme from a dropdown list:

  • BrownSugar
  • DarkStar
  • DesertBlue
  • DesertBluer
  • DesertGreen
  • DesertRed
  • DesertYellow
  • ExperienceBlue
  • ExperienceGreen
  • ExperienceRoyale
  • LightGray
  • Silver
  • SkyBlue
  • SkyBluer
  • SkyGreen
  • SkyKrupp
  • SkyPink
  • SkyRed
  • SkyYellow

About

JGoodies was developed by JGoodies Software. The version included in this plugin is the last free version, later versions require a fee.

Here's a description that I copied from the JGoodies website:

Precise Windows Emulation
Our Windows look&feel focuses on a precise emulation on Windows XP, Vista, 7, 8, 8.1, and now Windows 10 in the following areas: menus, icons, colors, borders, fonts, font sizes, insets, and widget dimensions. It honors the screen resolution (low vs. high dpi) to adjust sizes, insets, and widget dimensions.
