Column Ruler plugin

The main purpose of this plugin is to provide a ruler above the text area which marks off columns, but it also provides the following features:

Features

  • Highlights the columns of the current selection (if there is one)
  • Displays the dimensions of the current selection (if there is one)
  • Displays number of selected characters in the status bar
  • add marks anywhere you want by double-clicking on ruler
  • Dynamic marks that follow caret and wrap column
  • Right-clicking lets you set the wrap mode, and edit/delete marks
  • Drag marks along the ruler, double click them to toggle their guide
  • Can indicate the next tab stop
  • Line Guides

If you're not using a fixed width font such as Courier the column ruler isn't going to do you much good. Also, Fractional Font Metrics should be off for proper alignment, though you may decide it's close enough with them on. Go to Utilities->Global Options->Text Area to change the fractional font metrics setting.
