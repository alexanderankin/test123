Column Ruler plugin

The main purpose of this plugin is to provide a ruler above the text area which marks off columns, but it also provides the following features:
  • Indicates caret column
  • Highlights the columns of the current selection (if there is one)
  • Displays the dimensions of the current selection (if there is one)
  • Marks the column where text is wrapped (if wrap is on)
  • Right-clicking lets you set the wrap mode
  • You can drag the wrap marker to set the buffer's wrap column

If you're not using a fixed width font such as Courier the column ruler isn't going to do you much good.

I know that when selecting multiple lines it only highlights the difference between the start and end columns. I don't think it would be very useful to highlight the entire ruler. Same thing applies to 'Select All'.
