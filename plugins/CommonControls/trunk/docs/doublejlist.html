DoubleJList

The DoubleJList is a highly customizable component that provides two lists with buttons to move selected items between the lists, plus a popup menu to allow adding and deleting items from the lists.

Features:

  • List are generic, so any sort of item can be listed. The only restriction is the items must implement Comparable. This is so the lists can be sorted.
  • Titles and tooltips can be set independently on the lists.
  • Automatic sorting of the lists.
  • Any number of action listeners can be added to the buttons to perform additional operations on moving items.
  • Any number of list selection listeners can be added to the lists to perform additional operations when the list selection changes.
  • Popup menu is optional. It has built-in "Add" and "Delete" items that can be hidden or shown as desired.
  • Custom cell renderers can be set for each list independently.

Basic usage is to create a new DoubleJList, call the various "set" methods, then call the "init" method.

The "Delete" item can be shown on the popup by calling the setShowDelete method. The "Delete" item has a default action that prompts the user for confirmation of the delete and will set the "canDelete" value appropriately. Other action listeners added to the "Delete" item should call the canDelete method before performing any action.

Additional items may be added to the popup menu by calling the addMenuItem method.

Here's a concrete example from the Console plugin: 

        dlist = new DoubleJList<String>();
        dlist.setLeftListData(inactiveShells.toArray());
        dlist.setRightListData(activeShells.toArray());
        dlist.setLeftLabel( "Available User Shells" );
        dlist.setRightLabel( "Active User Shells" );
        dlist.setLeftToolTipText( "Right click to delete selected items" );
        dlist.setRightToolTipText( "Right click to delete selected items" );
        dlist.setShowDelete(true);
        dlist.addDeleteListener( new ActionListener(){

                public void actionPerformed( ActionEvent ae )
                {
                    if (dlist.canDelete()) {
                        List<String> names = dlist.getDeleteSelectedValues();
                        for (String name : names)
                        {
                            ServiceManager.unregisterService(Shell.SERVICE, name);
                            jEdit.unsetProperty("console.userShells." + name + ".code");
                        }
                        reloadLists();
                        reloadDefaultList();
                    }
                }
            }
        );
        dlist.init();
        addComponent( dlist, GridBagConstraints.NONE );
